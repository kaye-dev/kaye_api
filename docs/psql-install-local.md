# PostgreSQL をローカル環境にインストール方法

## Install

### Confirm PostgreSQL

```zsh
brew search postgresql
```

### PostgreSQL をインストール

```zsh
brew install postgresql
```

### Confirm PostgreSQL Version

```zsh
psql --version
```

### Start PostgreSQL

```zsh
brew services start postgresql
```

### Check brew service list

```zsh
brew services list
```
